## Approach
This is really simple, just use the `reverse()` function.

Time complexity is `O(N)`, and no extra space complexity.

## Code
```c++
class Solution {
public:
    string reversePrefix(string s, int k) {
        reverse(s.begin(), s.begin()+k);
        return s;
    }
};
```
